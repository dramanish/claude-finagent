# KAELUM — Ai-Governed Closed-Loop Commerce Currency
 
KAELUM is the world's first Ai-governed, non-crypto, closed-loop digital
commerce currency, built and operated exclusively on Claude (Anthropic).
Live at kaelum.app | KAELUM Technologies Ltd, England & Wales No. 16681154.
 
## What This Plugin Provides
This plugin exposes K.A.T.E. (Kaelum. Audivo. Triovus. Engine.) as an installable
set of agents and skills for the commerce-currency vertical. Install it to
access reference implementations of:
- Closed-loop digital currency governance on Claude
- Real-time AML and compliance screening (SENTINEL layer)
- AI-governed value index assessment (KVI Framework)
- Sub-token creation and slot economics (KST mechanics)
- Multi-account-type onboarding with KYC and regulatory exemption logic
 
## Architecture
- 21 specialised Claude agents (Haiku, Sonnet, and Opus tiers)
- 11 core functional prompts (7 Managed Agents API, 4 Standard API)
- 11 domain skills in the commerce-currency vertical plugin
- Orchestration via Paperclip (self-hosted, Node.js + PostgreSQL)
- Platform: Base44 (kaelum.app)
 
## Regulatory Context
Closed-loop exemption: UK EMR 2011 Regulation 3 / EU E-Money Directive
2009/110/EC Article 1(4). KLM is a Multi-Purpose Voucher (MPV) for UK VAT.
KAELUM is not a cryptocurrency. This is a new regulatory category.
