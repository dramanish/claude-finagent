# K.A.T.E. Commerce Currency Slash Commands

/klm-check        — Check KLM balance and active KLMback rate for a user
/kst-status       — Check KST sub-token performance and slot status
/kvi-score        — Return current KVI Health Score and dimension breakdown
/sentinel-score   — Return SENTINEL risk score for a specified transaction
/compliance-check — Run regulatory language check on a block of text
/merchant-perf    — Pull merchant performance summary for a given merchant ID
/creator-perf     — Pull creator performance summary for a given creator ID
/customer-summary — Pull spending and KLMback summary for a given customer ID
/distribution     — Calculate next KPR distribution estimate for a user
